# odin-recipes
a basic recipe website with few recipes

## Features
-The project consist of an index page where an intro to website and options between multiple dishes are given.
-With each option a preview image and description is given
-Upon clicking on any dish a new tab opens with ingrediants required and steps for making given.


## Tech Stack
-HTML
-Git, Github, Chrome
-Allrecipes.com(image and recipe)
